# Typography

Netflix Sans font family with system fallbacks.

```
font-family: Netflix Sans, Helvetica Neue, Helvetica, Arial, sans-serif;
```

Defined in `Typo.res`. All styles are Emotion class names.

- **400** — Regular: body, captions, headlines
- **500** — Medium: buttons, subtitles, emphasized text
- **700** — Bold: hero titles, section headers
