# Typography

Arteflix uses a clean, modern sans-serif font stack similar to Netflix.

## Font Family

```css
font-family: 'Netflix Sans', 'Helvetica Neue', Helvetica, Arial, sans-serif;
```

## Font Weights

| Weight | Value | Usage |
|--------|-------|-------|
| Regular | 400 | Body text, descriptions |
| Medium | 500 | Buttons, labels |
| Semibold | 600 | Card titles |
| Bold | 700 | Headings, hero text |

## Text Colors

| Color | Value | Usage |
|-------|-------|-------|
| Primary | `#ffffff` | Main text |
| Secondary | `rgba(255,255,255,0.7)` | Descriptions |
| Muted | `rgba(255,255,255,0.5)` | Metadata |
| Accent | `rgb(229, 9, 20)` | Links, highlights |
