## Layout

- Side padding: 4% viewport
- Section gap: 48px (`Spacing.xxl`)
- Card gap: 8px (`Spacing.sm`)

| Token | Value |
|-------|-------|

| `Radius.md` | 8px |
| `Radius.lg` | 16px |
| `Radius.full` | 9999px |
