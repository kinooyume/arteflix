## Common Patterns

### Page Layout
- **Side padding**: 4% of viewport width (responsive)
- **Section gap**: 48px between content zones
- **Card gap**: 8px between movie cards

### Components
- **Button padding**: 12px 24px
- **Card border-radius**: 4px
- **Modal padding**: 24px

## Responsive Breakpoints

| Name | Width | Usage |
|------|-------|-------|
| Mobile | < 600px | Single column, stacked layout |
| Tablet | 600px - 1024px | 2-3 columns |
| Desktop | 1024px - 1440px | Full layout, 4-6 cards |
| TV | > 1440px | Large cards, 6+ columns |
