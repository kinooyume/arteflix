# Arteflix UI

A Netflix-inspired component library for [Arte.tv](https://arte.tv) content, built with **ReScript** and **React**.

## Overview

This design system provides all the building blocks needed to create a streaming video interface:

- **Atoms**: Buttons, Tags, Icons
- **Molecules**: Cards, Previews, Episode blocks
- **Organisms**: Headers, Hero sections, Movie blocks
- **Templates**: Page layouts

## Tech Stack

| Technology | Purpose |
|------------|---------|
| ReScript | Type-safe UI components |
| React 19 | Component framework |
| Emotion | CSS-in-JS styling |
| Storybook | Component documentation |

### Blocks

### Navigation
