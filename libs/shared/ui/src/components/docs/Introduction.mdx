# Arteflix UI

Component library for the arteflix project. Built in ReScript + React.

## Structure

We loosely follow atomic design:
- **Atoms** — buttons, tags, small stuff
- **Molecules** — cards, previews
- **Organisms** — headers, hero sections, carousels
- **Templates** — full page layouts
